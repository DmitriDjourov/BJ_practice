### quick buttons

1.  Ctrl + Space — стандартное автодополнение кода;
2.  Ctrl + Shift + Space — умное автодополнение кода;
3.  Двойной Shift — искать везде;
4.  Alt + Enter — показать возможные варианты исправления;
5.  Alt + Ins — генерация кода (геттеры, сеттеры, конструктор и т.д.);
6.  Ctrl + P — информация о параметре (применяется на аргументе метода при вызове);
7.  Ctrl + W — расширить область выделения;
8.  Ctrl + Shift + W — уменьшить область выделения;
9.  Ctrl + E — список недавних файлов;
10. Shift + F6 — переименовать (переменную, функцию и т.д.);
11. Ctrl + R — сделать замену;
12. Ctrl + Q — быстрый просмотр документации;
13. Ctrl + / — закомментировать/раскомментировать строку;
14. Ctrl + Shift + / — закомментировать/раскомментировать блок кода;
14. Ctrl + Alt + I — автоматически добавить нужные отступы для строки/строк;
16. Ctrl + Shift + A — найти действие;
17. Shift + Alt + ↑ — переместить выбранные строки вверх;
18. Shift + Alt + ↓ — переместить выбранные строки вниз;
19. Ctrl + Shift + F7 — показать все упоминания в файле;
20. Ctrl + Alt + L — реформатировать код.
